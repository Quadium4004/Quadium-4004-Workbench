# Quadium 4004 Workbench

Educational Windows desktop workbench for the Intel MCS‑4 (4004): assembler, QuadBasic runtime, docs, and project tools.

Quadium 4004 Workbench is a Windows desktop workbench for exploring and programming an educational emulation of the Intel MCS‑4 (4004) era, with assembler tooling, the QuadBasic language and runtime, built‑in hardware documentation, and project workflow helpers.

---

## 🛠️ Overview

Quadium 4004 Workbench is a professional-grade simulation environment designed for computer historians, electronic engineers, and low-level programming enthusiasts. Developed as a solo project with the technical rigor of a senior engineer, its purpose is to preserve and explore the complete MCS-4 ecosystem (CPU, RAM, ROM, and I/O) exactly as conceived in 1971.

### Key Engineering Features:

* **Transparent Logic:** Zero black boxes. Monitor every register, bus, and memory cell in real-time. Analyze the internal operations of the world's first microprocessor with absolute visual clarity.
* **QuadBasic Transpiler:** Bridge the generational gap. Program the 4004 using a high-level syntax that translates directly into native MCS-4 machine code, making the 4-bit architecture accessible without losing technical depth.
* **Hardware Patching & Peripherals:** Configure your laboratory with a catalog of interactive modules. Connect virtual 7-segment displays (BCD/Mask), 8x8 LED matrices, LCDs, Pulse Generators, DIP switches, and ASCII keyboards to simulate a complete physical testbench.
* **Modern Export Capabilities:** Integrate 1971 logic with modern workflows. Redirect emulated output directly to standard `.txt` files on your host PC for external data logging and analysis.

### Multi-Instance Modular Workspace:

Designed for serious experimentation, the workbench offers a fully flexible interface:
* **Multiple Floating Windows:** Open several instances of RAM or ROM modules to monitor data flow across the entire system simultaneously.
* **Custom Aesthetics:** Toggle instantly between high-visibility modern layouts and classic CRT aesthetics (Retro-Amber or Phosphor-Green).

### Comprehensive Learning Blueprint:

The workbench includes an extensive library of built-in modules designed to guide the user from foundational logic gates to complex system interfacing, ensuring a deep understanding of early microprocessor architecture.

> *"A personal tribute from a senior engineer to the pioneers of 1971. Built for those who find beauty in the raw logic of the silicon that started it all."*

## 💻 Technical & Build Information

* **Development Environment:** Visual Studio (VB.NET, targeting .NET Framework as specified in the solution).
* **Build Instructions:** Open the Visual Studio solution file and build the main `Quadium 4004 Workbench` project.
* **Application Documentation:** User-facing documentation shipped with the application lives under `Quadium 4004 Workbench/UI/Help/`. It is written in English and embedded as resources for the built-in help windows.
